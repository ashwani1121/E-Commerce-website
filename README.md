# E-Commerce-website
A simple and responsive e-commerce website built using HTML, CSS, and JavaScript. It includes product listings, cart functionality, and interactive UI elements.

This project is a fully responsive E-Commerce website developed using HTML, CSS, and JavaScript. It includes dynamic product cards, add-to-cart functionality, and an interactive user interface designed to enhance the shopping experience. The goal of this project is to demonstrate core front-end development concepts such as UI/UX design, responsive layouts, DOM manipulation, animations, and client-side storage using JavaScript.
